# FlowMonkey

Minimal workflow execution engine.

## Quick Start

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test

# Build all packages
pnpm build
```

## Packages

| Package | Description |
|---------|-------------|
| [@flowmonkey/core](./packages/core) | Core execution engine |

## Development

```bash
# Watch mode for development
pnpm dev

# Run tests in watch mode
pnpm test:watch

# Type check
pnpm typecheck
```
